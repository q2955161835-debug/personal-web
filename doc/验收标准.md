# 验收标准

## 测试环境
- 操作系统：Windows 11，本地 PowerShell。
- Node.js：v22.17.1，npm 10.9.2。
- 浏览器：Chrome / Edge / Firefox / Safari 最新版（需支持 WebGL2）。
- 工作区：`d:\0文件夹\个人站`。

## 验收项 A：项目骨架与本地启动
- 验收步骤：
  1. 运行 `npm install` 安装依赖。
  2. 运行 `npm run dev` 启动开发服务器。
  3. 用 Edge 或 Chrome 打开 `http://127.0.0.1:5173/`。
  4. 等待 1 秒后 loader 隐藏，看到星空背景。
  5. 运行 `npm run typecheck` 确认无类型错误。
- 预期结果：
  - `node_modules/` 安装成功，无致命错误。
  - 开发服务器在 127.0.0.1:5173 启动。
  - 页面加载后 loader（橙色脉冲圆）在 1 秒内淡出。
  - 全屏黑色背景上出现三层星空（远景密集小白点 + 中景彩色星点 + 近景闪烁亮星）。
  - 鼠标移动时相机有轻微视差。
  - 顶部导航显示 "FAN JUNJIE · 范俊杰" 和 GitHub / Contact 链接。
  - 底部显示滚动提示 "SCROLL TO EXPLORE" 和 HUD 信息。
  - `npm run typecheck` 退出码 0。
- 结果记录：
  - 2026-07-05：通过。`npm install` 成功；`npm run dev` 启动成功；`npm run typecheck` 退出码 0；`npm run build` 构建成功（three 504kB / index 20.5kB）。视觉验收由用户在浏览器确认。

## 验收项 B：入口恒星首页（阶段2）
- 验收步骤：
  1. 浏览器打开 `http://127.0.0.1:5173/`。
  2. 等待 loader 淡出，观察首页中心是否出现橙红色太阳球体。
  3. 检查太阳表面是否有湍流噪波动画（黑子分层 + 颜色 core 黄白 / mid 橙 / edge 红橙）。
  4. 检查太阳外层是否有日冕半透明辉光（1.35x）和远场辉光（1.8x）。
  5. 检查太阳前方是否漂浮紫蓝双色倒V Logo（中间隔断，顶端间隙可见）。
  6. 检查左下角是否显示 Hero 文字："FAN JUNJIE" 渐变大字 + "范俊杰" 字距小字 + "AI 产品 / 数据分析实习" + "苏州 · 盐城师范学院 · GitHub 主页"。
  7. 移动鼠标，检查是否生成蓝-紫-粉色调的流星粒子拖尾（沿移动反方向喷射，渐隐消失）。
  8. 检查 Logo 是否有轻微悬浮旋转和发光强度脉动。
  9. 检查太阳是否缓慢自转。
  10. 运行 `npm run typecheck` 确认无类型错误。
- 预期结果：
  - 太阳球体居中可见，表面有动态湍流噪波和颜色分层。
  - 日冕和远场辉光呈半透明橙红色光环。
  - 倒V Logo 紫蓝双色，中间隔断清晰可见，轻微悬浮旋转。
  - Hero 文字渐变显示，左下角定位，1.4s 淡入。
  - 鼠标移动有蓝-紫-粉流星拖尾，180 粒子限流不卡顿。
  - `npm run typecheck` 退出码 0。
- 结果记录：
  - 2026-07-05：自动验证通过。`npm run typecheck` 退出码 0；`npm run build` 构建成功。视觉项（1-9）需用户人工确认。
