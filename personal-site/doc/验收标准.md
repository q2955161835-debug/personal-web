## 全局浏览器纵览验收

### 验收步骤
1. 打开 `http://localhost:3000`。
2. 从 Hero 首屏开始，慢速滚动经过 About、Projects、Data Analysis、Experience、Contact 到页面底部。
3. 从 Contact 向上滚回 Projects、About 和 Hero。
4. 在滚动期间截图或观察关键状态，并检查浏览器控制台新增错误。

### 预期结果
- Hero 粒子在首屏正常显示，从 Projects 回滚到上方页面后不消失。
- About 内容与全局 Canvas 不互相遮挡，页面上下没有大段失衡空白。

- 滚动过渡连续自然，项目切换节奏跟随鼠标滚轮速度，不出现离散跳变过快。
- Data Analysis、Experience、Contact 段上下衔接紧凑，不出现大段空白或内容漂浮失衡。
- Canvas 始终非空，无主要视觉对象飞出视口或遮挡正文。

## Projects DNA 项目段验收

### 验收步骤
1. 滚动到 Projects 段。
2. 慢速滚动观察 10 个项目的切换。
3. 点击右侧项目序号跳转到首尾与中间项目。
4. 点击 DNA 键标签或右侧项目序号，检查是否定位到对应项目。
5. 从 Projects 上划回 About，再从 About 下滑进入 Projects，观察 DNA 是否反向消散/重新聚合。
6. 观察项目标签、右侧项目说明、DNA 主体、背景粒子关系。

### 预期结果
- 项目名称不重叠，DNA 键位贴近对应项目位置，不堆在底部。
- 左侧状态栏只显示章节与当前序号，不出现“当前 DNA 键位”旧描述，不重复显示 DNA 键位上的项目标题和副标题。
- 右侧序号导航可点击，激活态与当前项目一致。
- DNA 键标签跟随同一螺旋变换，位置固定在对应键附近，点击标签可平滑定位到该项目。
- 当前项目说明直接固定在右侧，展示项目定位、技术栈、标签和 GitHub 链接，随滚动切换淡入淡出，不再打开旧项目详情弹层。
- DNA 主体足够长，首个项目键位进入时已经位于 DNA 结构上，不作为过渡散点使用。
- DNA 上端必须有 0 号空槽段用于粒子聚合动画，0 号段没有项目内容；第 1 个真实项目只能在聚合完成后进入居中展示。
- DNA 末端需要保留到第 11 个空槽的位置；第 10 个真实项目必须有与其他项目接近的完整居中展示长度。
- 从 About 下滑进入 Projects 时，粒子由散点聚合成 DNA；从 Projects 上划回 About 时，DNA 反向消散且不残留干扰 About。
- DNA 除项目粗碱基对外，必须有较细碱基对填充，背骨线条有宽度但不能固定点过曝。
- 相机角度保持稳定，滚动驱动 DNA 自身旋转、向下移动、粒子聚合与项目键缩放。
- 鼠标划过 DNA 粒子时粒子被打散，鼠标移开后回弹复原。
- DNA 主体清晰，后景粒子数量克制，不干扰项目浏览。
- 从 Projects 切回上方页面后，首页粒子和相机状态恢复正常。
- 右侧说明与 DNA 标签不能放重复大段内容；DNA 内部编号标记可作为键位提示，但不能遮挡项目名。
- 右侧说明切换必须滞后于 DNA 键位运动：当前项目键先到达居中焦点，右侧说明再切换为该项目。

## Data Analysis 数据分析作品集验收

### 验收步骤
1. 滚动到 `#data-analysis`。
2. 观察页面是否 pin 固定，继续滚轮下滑。
3. 检查柱状图是否随滚轮横向移动，且项目按含金量从左到右排序。
4. 点击至少两根玻璃柱，检查左侧详情是否切换。
5. 检查常规坐标轴、横轴均衡器竖线、方法星云和背景星点是否可见且不遮挡正文。

### 预期结果
- 标题、说明、悬浮详情、常规坐标轴、短彩色竖线波形和 `fluid-glass` 柱体完整可读，详情不得压住主体柱状图。
- 玻璃柱体有不同颜色和真实流体玻璃质感，hover/鼠标移动产生受限 3D 倾斜反馈。
- 柱体高度呈明显正态分布，中间区域高、两端低，并保留局部随机起伏；不得出现大部分柱体近似等高。
- 柱体上不显示含金量数字；排序仍按 `impactScore` 从左到右。
- 滚动进度与点击柱体都能切换详情，详情标题、核心、方法、工具和关键发现同步更新；不得显示时间、交付文件数量或 `价值点` 旧文案。
- 滚动在该区先推动柱状图横向移动，不能直接跳过整个 section。
- 坐标轴必须包含正常水平轴、左侧纵轴和左侧短横线；水平轴上生成等间距细竖线，竖线高度随滚动变化，当前展示柱体下方形成类似 audio waveform visualization 与 equalizer bars 的局部波峰，不使用旧粒子坐标轴。
- 从 Projects 进入 Data Analysis 后，DNA 必须在柱状图开始横向运动时快速逸散，不能长时间残留在 Data 背景里。
- DNA 消散必须从第 10 个真实项目到第 11 个空槽的中点之后才开始，到达 Data Analysis pinned 起点时完成。
- 方法论图谱改为星云：默认方法名隐藏，当前项目涉及的方法粒子变大并显示方法名；不使用旧卡片 tab 过滤形态。
